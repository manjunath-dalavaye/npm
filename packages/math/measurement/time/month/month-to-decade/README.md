# @ilihub/month-to-decade

TypeScript functions for converting month to decade.

![NPM Version](https://img.shields.io/npm/v/%40ilihub%2Fmonth-to-decade?color=33cd56&logo=npm)
![NPM](https://img.shields.io/npm/l/%40ilihub%2Fmonth-to-decade)
[![CodeFactor](https://www.codefactor.io/repository/github/ilihub/npm/badge)](https://www.codefactor.io/repository/github/ilihub/npm)
[![Socket Badge](https://socket.dev/api/badge/npm/package/@ilihub/month-to-decade)](https://socket.dev/npm/package/@ilihub/month-to-decade)

## Installation

```bash
# if you're using pnpm
pnpm add @ilihub/month-to-decade

# or, if you're using npm
npm install @ilihub/month-to-decade

# or, if you're using yarn
yarn add @ilihub/month-to-decade
```

## Usage

```javascript
import { MonthToDecade } from "@ilihub/month-to-decade";

const month = 1200;

const decade = MonthToDecade(month);
console.log(decade); // 10
```
