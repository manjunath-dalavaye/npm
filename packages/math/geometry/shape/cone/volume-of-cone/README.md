# @ilihub/volume-of-cone

A TypeScript library for calculating the volume of a cone.

![NPM Version](https://img.shields.io/npm/v/%40ilihub%2Fvolume-of-cone?color=33cd56&logo=npm)
![NPM](https://img.shields.io/npm/l/%40ilihub%2Fvolume-of-cone)
[![CodeFactor](https://www.codefactor.io/repository/github/ilihub/npm/badge)](https://www.codefactor.io/repository/github/ilihub/npm)
[![Socket Badge](https://socket.dev/api/badge/npm/package/@ilihub/volume-of-cone)](https://socket.dev/npm/package/@ilihub/volume-of-cone)

## Installation

```bash
# if you're using pnpm
pnpm add @ilihub/volume-of-cone

# or, if you're using npm
npm install @ilihub/volume-of-cone

# or, if you're using yarn
yarn add @ilihub/volume-of-cone
```

## Usage

```javascript
import { VolumeOfCone } from "@ilihub/volume-of-cone";

const radius = 5;
const height = 13;

const volume = VolumeOfCone(radius, height);
console.log(volume); // 340.3392041388942
```
